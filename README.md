# 🎙️ Tone Weaver

> AI-powered voice transformation and accent conversion web app

---

## 📌 About

Tone Weaver is my Final Year Project — a full stack web application that allows users to transform their voice and convert accents using AI-powered audio processing.

---

## ✨ Features

- 🎤 Real-time voice transformation
- 🌍 Accent conversion
- 🔊 Audio comparison (original vs converted)
- ☁️ Cloud storage for audio files
- 👤 User authentication and history

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- TypeScript
- Tailwind CSS

**Backend:**
- FastAPI (Python)
- REST APIs

**Database & Cloud:**
- MongoDB
- AWS S3

**Audio Processing:**
- librosa
- pydub
- praat-parselmouth

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/muhammadabdullahchauhan/tone-weaver

# Install frontend dependencies
cd frontend
npm install
npm run dev

# Install backend dependencies
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
